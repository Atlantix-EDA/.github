### Robust and Modular Software for Electronic Design Automation
*Automating the future of electronics* · [Docs](https://docs.copperforge.dev) · atlantix-eda@proton.me

We build modern, open-core tooling that makes hardware design faster and more
reliable — from KiCad library generation through PCB release and manufacturing,
along with FPGA design automation and tooling. 

### 🛠 Flagship Projects

| Project | What it does |
|---|---|
| **[CopperForge](https://github.com/Atlantix-EDA/CopperForge)** | Companion PCB release & manufacturing tool for KiCad — DFM, gerber/BOM, 3D. |
| **[atlantix-eda](https://github.com/Atlantix-EDA/atlantix-eda)** | Programmatically generated PCB libraries for robust product design. |
| **[grafica-ide](https://github.com/Atlantix-EDA/grafica-ide)** | DSL-driven graphics IDE — scriptable canvas + node-graph editing. |
| **[quarri](https://github.com/Atlantix-EDA/quarri)** | Linux launcher for Intel/Altera Quartus, dark theme + multi-install. |
